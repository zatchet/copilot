GitHub Copilot: An Overview

What is Copilot?

Released in 2021 by Microsoft and OpenAI, GitHub Copilot is an AI pair programmer that helps you write code faster and with less effort. It is powered by OpenAI's Codex, a machine learning model trained on a diverse range of public code repositories. Copilot can generate code suggestions, write entire functions, and even complete entire lines of code based on the context of your project.

How does Copilot work?

Copilot works by analyzing the code you've already written as well as in-line comments to generate code suggestions. It uses machine learning to predict what code you might want to write next and offers suggestions in real-time as you type.



What are the benefits?

  • Productivity increase: Research has shown that programmers can complete tasks 55.8% faster when using Copilot.
  • Learn new programming languages: Copilot can be useful tool to learn new programming languages by providing code examples and suggestions.

What are the drawbacks?

  • Correctness: Copilot generated solutions to problems are not always functionally correct.
  • Code quality: Copilot may generate code that is not effficient and/or does not adhere to common design principles and programming paradigms.
  • Security: Copilot may generate code that is vulnerable to security threats, such as SQL injection or cross site scripting.
  • Dependency on AI: Copilot may lead to a dependency on AI for coding tasks, reducing the need for developers to learn and understand programming concepts.
  • Privacy concerns: Copilot may raise privacy concerns, as it analyzes your code and project context to generate suggestions.
